## Recipe handbook

### User stories

```
As a user
So that I can find the recipe I want
I want to be able to search for a recipe.
```

```
As a user
If I have a varying amount of guests for food
I want to be able to update the number of servings
```

```
As a user
If I like a particular recipe
I want to be able to bookmark specific recipes
```

```
As a user
So that I can customise the food I make
I want to be able to make my own recipes
```

```
As a user
So that I can come back to my recipees and bookmarks
I want to be able to leave the app and come back with all my information saved
```
